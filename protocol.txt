client -> server :
	username -> connection init				|	conn_init <file_descriptor> <username>
	ask for eligible users to play with		|	av_users <file_descriptor>
	which player you wanna play with		|	play_with <file_descriptor> <username>
	game request/confirmation/decline		|	game_request <file_descriptor> <response> (0 for decline | 1 for accept | 2 for request)
	game info (move to be made) : A 1 A 4	|	make_move <file_descriptor> <init_x> <init_y> <final_x> <final_y>
	end connection request/forfeit			|	forfeit <file_descriptor>
	end game (ckeck mate)					|	end_game <user>

	
server -> client : 
	list with other users					|	av_users <file_descriptor> [<username>,<username>,<username>]
	send color info							|	set_color <file_descriptor> <color> (1 for black / 0 for white)
	game request/confirmation/decline		|	game_request <file_descriptor> <response> (0 for decline | 1 for accept | 2 for request) <user> (for request)
	game info (move to be made) : A 1 A 4	|	make_move <file_descriptor> <init_x> <init_y> <final_x> <final_y>
	disconcetion error (from oponent)		|	player_left <file_descriptor> <usernameP2>
	illegal move							|	illegal_move <file_descriptor>
	username ok								|	conn_init <response> (0 for decline | 1 for accept)
